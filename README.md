SOP 文档管理系统 功能介绍

一、基础介绍

本系统为轻量化 SOP 流程文档管理平台，采用文件化存储、免数据库部署，支持分部门、分岗位归类管理流程文件，搭配权限管控、检索查询、文档编辑，适合内部标准化流程查阅与维护。

二、全部常用功能

匿名浏览开关后台可自由开启 / 关闭游客匿名访问，关闭后必须登录才能查看 SOP 文档，保障内容安全。

多用户权限管理

支持管理员手动新建、删除系统账号；

双权限角色：管理员（全部操作权限）、普通用户（仅浏览查看，无编辑 / 删除 / 后台管理权限）。

账号登录与退出提供登录页面，账号密码验证登录，支持一键退出登录。

全局搜索检索顶部自带搜索功能，可通过关键词模糊检索 SOP 标题、正文内容，快速定位需要的流程文档。

分类栏目管理

一级部门管理：自定义新增、编辑、删除部门分类；

二级岗位管理：绑定对应部门，按岗位划分 SOP，左侧菜单栏折叠展开，分类清晰。

SOP 文档管理（管理员专属）支持新增、编辑、删除 SOP 文档，可绑定对应部门与岗位，方便归类归档。

富文本内容编辑内置简易编辑器，支持文字加粗、多颜色字体标记、清除格式，满足流程重点标注需求。

文档在线浏览所有 SOP 卡片化展示，支持点击查看全文，排版整洁，适合日常快速查阅。

系统基础设置可自定义修改系统网站名称，个性化配置站点信息。

操作日志记录自动记录登录、文档修改、配置更改等操作日志，便于后期追溯查看。

适配移动端 / 电脑端页面布局简洁自适应，搭配回到顶部快捷按钮，日常浏览操作便捷。
